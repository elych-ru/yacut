# Проект yacut - укоротитель ссылок

### Подготовка к запуску:
- Установить зависимости (в предварительно активированное виртуальное окружение):
```
python3 -m pip install --upgrade pip
pip install -r requirements.txt
```
- Создать и заполнить .env файл с настройками на основе предоставленного .env.sample
- Создать пустую БД:
```
flask create_db
```
### Запуск проекта
```
flask run
```

### Вспомогательные команды
- Создать пустую БД
```
flask create_db
```
- Очистить (пересоздать) БД
```
flask flush_db
```
